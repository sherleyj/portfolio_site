Scrapiic
Scrapiic is a Node app that uses Puppetteer to scrape a webpage for all of its pictures. After working on Kado, I was inspired to create Scrapiic when I heard about headless Chrome. When Kado scrapes a product's page for images (it uses Beautiful Soup for this), it didn't always include the product's image for pages that are loaded dynamically, or any images. Headless Chrome gets around this by loading the page as a browser would (because it is a browser) before attempting to scrape the page. This makes it kinda slow but it's pretty cool... Puppeteer is the Node library that provides the API to control headless Chrome and was utilized in this project.
